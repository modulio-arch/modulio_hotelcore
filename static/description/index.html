Hotel Core

Sistem Manajemen Hotel Dasar untuk Odoo

🏨 Apa itu Hotel Core?

Hotel Core adalah modul dasar yang menyediakan fondasi untuk mengelola hotel Anda. Seperti fondasi rumah, modul ini harus ada sebelum Anda bisa menggunakan fitur-fitur hotel lainnya seperti sistem reservasi, housekeeping, atau penagihan.

📋 Manajemen Jenis Kamar

Atur berbagai jenis kamar di hotel Anda dengan mudah:

  • Buat kategori kamar (Standard, Deluxe, Suite, dll)
  • Tentukan kapasitas maksimal tamu per kamar
  • Set harga dasar untuk setiap jenis kamar
  • Catat fasilitas yang tersedia (AC, WiFi, TV, dll)
  • Atur ukuran kamar dalam meter persegi

🏠 Manajemen Kamar Individual

Kelola setiap kamar di hotel Anda secara detail:

  • Buat daftar semua kamar dengan nomor dan lantai
  • Kaitkan setiap kamar dengan jenis kamar yang sesuai
  • Lacak status kamar secara real-time
  • Catat catatan khusus untuk setiap kamar
  • Flag kamar yang memerlukan perbaikan

🔄 Sistem Status Kamar

Pantau status setiap kamar dengan sistem yang terorganisir:

Alur Status Kamar:

Kosong & Siap → Dipesan → Terisi → Kotor → Diperiksa → Kosong & Siap
  • Status "Tidak Beroperasi" untuk kamar yang sedang diperbaiki
  • Catat waktu dan alasan setiap perubahan status
  • Validasi otomatis untuk memastikan alur yang benar

👥 Manajemen Akses Staf

Atur siapa yang bisa mengakses apa di sistem hotel Anda:

  • Staf Hotel: Bisa melihat dan mengubah status kamar, tidak bisa membuat kamar baru
  • Manajer Hotel: Akses penuh untuk mengelola semua aspek hotel
  • Setiap perubahan dicatat untuk audit dan keamanan

💬 Komunikasi Internal

Fasilitas komunikasi untuk tim hotel Anda:

  • Kirim pesan internal terkait kamar atau jenis kamar
  • Buat tugas dan follow-up untuk maintenance
  • Lacak semua perubahan data secara otomatis
  • Riwayat lengkap untuk referensi masa depan

🚀 Mengapa Hotel Core Penting?

Hotel Core adalah modul wajib yang harus diinstall sebelum modul hotel lainnya seperti:

  • Sistem Reservasi - untuk booking kamar
  • Housekeeping - untuk manajemen cleaning
  • Front Office - untuk check-in/check-out
  • Sistem Penagihan - untuk billing tamu

Tanpa Hotel Core, modul-modul tersebut tidak bisa berfungsi karena mereka membutuhkan data kamar dan jenis kamar yang didefinisikan di sini.

📊 Dashboard & Laporan

Pantau operasional hotel Anda dengan mudah:

  • Dashboard utama menampilkan status semua kamar
  • Filter kamar berdasarkan status, lantai, atau jenis
  • Laporan jumlah kamar per jenis
  • Tracking maintenance dan perbaikan

Siap Memulai?

Hotel Core memberikan fondasi yang kuat untuk mengelola hotel Anda. Install modul ini terlebih dahulu, lalu tambahkan modul hotel lainnya sesuai kebutuhan bisnis Anda.
